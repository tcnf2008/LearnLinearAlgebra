08. SVD 与 PCA

本页把 SVD 和 PCA 放在同一条直觉线上：SVD 说明任意矩阵如何把单位圆变成椭圆，PCA 则在数据云里寻找方差最大的方向，并把数据投影到这个方向上。

本章要解决的问题

核心问题

一个复杂矩阵变换或一团高维数据，能不能拆成少数几个最重要的正交方向？

  1. SVD：任意矩阵都可以写成 A=UΣV^T。
  2. 几何：V^T 先换输入方向，Σ 沿坐标轴缩放，U 再换输出方向。
  3. 奇异值：σ1, σ2, ... 是单位球被拉伸成椭圆或椭球后的半轴长度。
  4. 低秩近似：只保留最大的奇异值，就保留了最强的变换方向，舍弃弱方向。
  5. PCA：对中心化数据，协方差矩阵的特征方向就是 principal components。
  6. 降维：把数据投影到前几个主成分上，可以用较少坐标保留最多方差。

知识讲解

1. SVD 是什么

SVD 把任意矩阵拆成三个部分：

A = U Σ V^T

V^T 选择输入中的正交方向，Σ 沿这些方向缩放，U 把结果放到输出空间的正交方向上。

2. 奇异值表示什么

把单位圆喂给矩阵，输出通常是一条椭圆。椭圆的长半轴和短半轴长度就是奇异值 σ1、σ2。

如果某个奇异值接近 0，说明矩阵几乎把某个方向压没了；如果它正好是 0，矩阵的 rank 会下降。

3. 为什么看 A^T A

右奇异向量来自 A^T A 的特征向量：

A^T A v_i = σ_i^2 v_i

因为 ||Av||^2 = v^T A^T A v，所以寻找被拉得最长的输入方向，等价于研究 A^T A。

4. 低秩近似

如果只保留最大的 k 个奇异值，就得到 rank k 的近似矩阵：

A_k = U_k Σ_k V_k^T

直觉上，这保留了最强的拉伸方向，舍弃弱方向。在图像压缩、推荐系统和噪声过滤里都很常见。

5. PCA 为什么是投影

PCA 先把数据中心化，再找让投影方差最大的方向。第一主成分是数据云最“长”的方向。

Cov = X_centered^T X_centered / n

协方差矩阵的特征向量给出主成分，特征值给出沿这些方向的方差。

6. SVD 和 PCA 的关系

对中心化数据矩阵 X 做 SVD：

X = U Σ V^T

V 的列就是 PCA 的主成分方向，Σ^2 与各主成分解释的方差成正比。PCA 可以看成 SVD 在数据分析中的一个核心应用。

本章最重要的一句话：SVD 找矩阵最强的输入/输出方向，PCA 找数据最强的方差方向；两者都在用正交方向压缩复杂结构。

思考题

  1. 为什么只保留最大的奇异值通常可以过滤掉弱信号或噪声，但也可能丢掉少数重要结构？

参考答案要点

展开精简答案
  1. 单位圆中每个方向长度都是 1；矩阵作用后，被拉得最长和次长的正交方向形成椭圆半轴，长度就是奇异值。
  2. 不中心化时，PCA 会优先解释数据整体离原点的位置，而不是数据自身的展开方向。
  3. 奇异值是 3 和 1，右奇异方向分别是 x 轴和 y 轴。
  4. 第一主成分是 x 轴方向；总方差是 4+1=5，第一主成分保留 4/5=80%。
  5. 最大奇异值对应能量或方差最大的结构，所以弱方向可能是噪声；但如果小方向承载罕见但关键的信息，截断就会丢失它。
