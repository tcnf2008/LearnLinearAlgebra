05. 行列式

本页把 determinant 从公式 ad-bc 拉回几何本质：一个矩阵把单位正方形变成平行四边形，行列式就是这个平行四边形的有向面积。

本章要解决的问题

核心问题

矩阵把整个平面拉伸、压缩、剪切或翻转时，面积会变成原来的多少倍？方向感是否被翻转？

  1. 对象：二维行列式先理解为矩阵变换对面积的有向缩放因子。
  2. 列向量：A 的两列是单位正方形两条边的落点。
  3. 面积：这两列张成的平行四边形面积绝对值是 |det(A)|。
  4. 符号：det(A)>0 保持方向感，det(A)<0 翻转方向感。
  5. 零：det(A)=0 表示面积被压成 0，平面退化成线或点，矩阵不可逆。
  6. 复合：连续做两个变换时，面积缩放因子相乘：det(AB)=det(A)det(B)。

知识讲解

1. 行列式是什么

对二维矩阵，行列式最先应该理解成面积缩放因子，而不是一个孤立公式。

A = [[a, b],
     [c, d]]

det(A) = a*d - b*c

矩阵把单位正方形变成由两列 [a,c]、[b,d] 张成的平行四边形。这个平行四边形的有向面积就是 det(A)。

2. 绝对值表示面积倍数

如果 |det(A)|=3，任何区域经过 A 后面积都会变成原来的 3 倍；如果 |det(A)|=0.5，面积变成一半。

这不只对单位正方形成立。线性变换会把网格中的每个小方格都以同样比例缩放面积。

3. 符号表示方向感

二维平面里，e1 到 e2 的顺序有一个方向感。矩阵作用后，如果第一列到第二列仍保持同样的逆时针顺序，行列式为正；如果顺序翻转，行列式为负。

  • det(A)>0：保持 orientation。
  • det(A)<0：发生一次镜像式翻转。

4. 为什么 det=0 表示不可逆

det(A)=0 意味着单位正方形被压成面积为 0 的形状：一条线或一个点。

这时二维信息丢失了。多个不同输入会落到同一个输出，所以无法唯一反推输入，也就没有逆矩阵。

5. 公式 ad-bc 的意义

ad-bc 可以看成两个方向贡献的有向面积差。更本质地说，它是二维列向量叉积：

[a, c] x [b, d] = a*d - c*b

正负号记录两列的先后方向，绝对值记录平行四边形面积。

6. 复合时为什么相乘

如果 B 先把面积放大 det(B) 倍，再由 A 放大 det(A) 倍，总面积缩放就是两者相乘。

det(AB) = det(A)det(B)

符号也会相乘：翻转一次为负，翻转两次又回到正。

本章最重要的一句话：行列式是矩阵变换对面积或体积的有向缩放因子；二维公式 ad-bc 是这个几何量的计算方式。

思考题

  1. 为什么旋转矩阵的行列式通常是 1，而镜像翻转的行列式是 -1？

参考答案要点

展开精简答案
  1. det=0 表示面积被压成 0，二维平面退化成线或点，信息丢失后无法唯一反推输入。
  2. 负号说明方向感被翻转；面积倍数是 |det(A)|。
  3. 2*4-1*3=5，面积变成原来的 5 倍，方向感保持。
  4. 1*4-2*2=0，不可逆；两列共线，单位正方形被压成一条线。
  5. 旋转只改变方向不改变面积，也不翻转 orientation，所以 det 是 1；镜像不改面积但翻转 orientation，所以 det 是 -1。

教学组织依据

本页采用“面积缩放 -> 符号 -> 零行列式 -> 公式 -> 复合”的顺序，让公式服务于几何意义，而不是反过来。
