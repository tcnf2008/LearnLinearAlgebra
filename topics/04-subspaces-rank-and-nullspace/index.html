04. 子空间、秩与零空间

本页把矩阵看成一个空间压缩器：它可能把整个平面保持为平面，也可能压成一条线，甚至压成一个点。column space 描述所有可能输出，rank 描述输出维度，null space 描述哪些输入被压到零。

本章要解决的问题

核心问题

一个矩阵到底保留了多少输出方向，又把哪些输入方向压没了？

  1. 子空间：一个集合如果包含零向量，并且对加法和数乘封闭，它就是 subspace。
  2. 列空间：Col(A) 是矩阵列向量所有线性组合组成的集合，也就是所有可能输出。
  3. 秩：rank(A) 是列空间的维数，表示矩阵输出还能覆盖多少独立方向。
  4. 零空间：N(A) 是所有满足 Ax=0 的输入，也就是被矩阵压到零的方向。
  5. 解方程：Ax=b 有解，当且仅当 b 落在 Col(A) 中。
  6. 维数守恒：对 2x2 矩阵，rank(A)+dim N(A)=2。

知识讲解

1. 子空间是什么

子空间不是“长得像一小块区域”的集合，而是仍然能做线性代数的集合。它必须满足：

0 在集合中
u, v 在集合中 => u+v 也在集合中
u 在集合中, k 是数 => k*u 也在集合中

二维平面里的典型子空间只有三类：原点、一条穿过原点的直线、整个平面。

2. Column space 是所有可能输出

矩阵乘向量可以看成列向量的线性组合：

A[x,y] = x*col1(A) + y*col2(A)

当 x,y 取遍所有实数，所有可能的输出就形成 Col(A)。所以 Ax=b 是否有解，本质上是在问：b 是否属于 Col(A)。

3. Rank 表示输出维度

rank(A) 是 column space 的维数。对 2x2 矩阵：

  • rank=2：两列不共线，输出覆盖整个平面。
  • rank=1：两列共线但不全为零，输出只落在一条线。
  • rank=0：两列都是零向量，所有输入都输出到原点。

4. Null space 是被压没的输入

N(A) 收集所有满足 Ax=0 的输入。它总是子空间，因为线性变换会保留加法和数乘。

N(A) = { x | Ax = 0 }

如果矩阵把平面压成一条线，那么一定有一条输入方向被压到零；如果矩阵满秩，只有零向量会被送到零。

5. 秩-零化度关系

输入空间里的维度不会凭空消失，它要么变成输出方向，要么被压进 null space。对二维输入：

rank(A) + nullity(A) = 2

满秩矩阵保留两个输出方向，所以 nullity 是 0；rank 1 矩阵只保留一个输出方向，所以 nullity 是 1。

6. 常见混淆

  • column space 在输出空间里，null space 在输入空间里。对方阵可以画在同一平面，但含义不同。
  • rank 不是矩阵里非零数字的个数，而是独立输出方向的个数。
  • 一条不过原点的直线不是子空间，因为它不包含零向量，也不对数乘封闭。
  • Ax=b 的解集不是 column space；column space 是所有可能的右侧 b。

本章最重要的一句话：column space 问“能输出到哪里”，null space 问“哪些输入被压到零”，rank 问“输出还剩几个独立方向”。

少量练习

概念题

  1. 为什么一条不经过原点的直线不是子空间？
  2. Col(A) 和 N(A) 分别生活在输入空间还是输出空间？它们各回答什么问题？

手算题

  1. 设 A=[[1,2],[2,4]]，求 rank(A)，并写出 N(A) 的一个方向。
  2. 设 A=[[1,0],[0,0]]，判断 b=[2,0] 和 b=[2,1] 是否在 Col(A) 中。

互动实验题

  1. 点击“满秩”“压成直线”“零矩阵”，观察 rank 和 nullity 如何变化。
  2. 切到“4 Ax=b”，在“投影到 x 轴”预设下拖动目标 b，找一个有解目标和一个无解目标。

思考题

  1. 为什么 rank 变小通常意味着某些不同输入会得到相同输出？

参考答案要点

展开精简答案
  1. 不过原点就不包含零向量；而且取线上一点做数乘，通常会离开这条线。
  2. Col(A) 在输出空间，回答“哪些 b 可以被表示成 Ax”；N(A) 在输入空间，回答“哪些 x 会被送到零”。
  3. 两列共线且非零，所以 rank(A)=1。解 x+2y=0，一个 null space 方向是 [-2,1]。
  4. [2,0] 在 column space 中；[2,1] 不在，因为这个矩阵只能输出到 x 轴。
  5. rank 分别是 2、1、0；对应 nullity 是 0、1、2。
  6. rank 变小表示某些输入方向被压到零；沿这些方向移动不会改变输出，所以不同输入可能落到同一个输出。

教学组织依据

本页采用“子空间封闭性 -> column space -> rank -> null space -> Ax=b”的顺序，目标是把抽象定义放回矩阵变换和解方程的语境中。
