01. 向量与线性组合

学习目标

线性代数关心的是线性结构。最基础的线性结构来自两个动作：相加和缩放。

  • 理解向量为什么可以表示位移、数据、状态或坐标。
  • 解释向量加法和数乘的几何意义。
  • 理解线性组合 a*u + b*v。
  • 理解 span 是所有线性组合形成的集合。
  • 区分线性相关和线性无关。
  • 理解基和坐标之间的关系。

知识讲解

1. 向量是什么

在二维平面中，向量可以画成从原点出发的箭头，例如 v = [3, 2] 表示向右走 3 个单位，再向上走 2 个单位。

但箭头只是直观模型。更一般地，向量可以表示位移、数据、状态或坐标。

从第一性原理看，向量最重要的能力是：两个向量可以相加，一个向量可以被数字缩放。

2. 向量加法

设 u = [u1, u2]，v = [v1, v2]，则：

u + v = [u1 + v1, u2 + v2]

几何意义是：先走 u，再走 v，最终位置就是 u + v。

u = [2, 1]
v = [1, 3]
u + v = [3, 4]

3. 数乘

设 v = [v1, v2]，a 是一个数字，则：

a * v = [a * v1, a * v2]
  • a > 1：沿原方向拉长。
  • 0 < a < 1：沿原方向缩短。
  • a = 0：变成零向量。
  • a < 0：反向并缩放。

4. 线性组合

如果有两个向量 u 和 v，那么：

a*u + b*v

叫做 线性组合（linear combination）。系数 a 和 b 控制你沿着两个方向分别走多少。

u = [2, 0]
v = [0, 3]
a = 1.5
b = 2
a*u + b*v = [3, 6]

5. Span

给定一组向量，所有可能的线性组合构成它们的 span。

span(u, v) = 所有 a*u + b*v

一个非零二维向量的 span 是穿过原点的一条直线。两个不共线二维向量的 span 是整个二维平面。

6. 线性相关与线性无关

如果一个向量可以由其他向量线性组合出来，这组向量线性相关（linearly dependent）。

u = [1, 2]
v = [2, 4]
v = 2u

这两个向量看起来有两个箭头，但只提供了一个方向。

7. 基与坐标

如果一组向量既能 span 整个空间，又线性无关，那么它就是这个空间的一组基（basis）。

e1 = [1, 0]
e2 = [0, 1]
[x, y] = x*e1 + y*e2

坐标不是向量本身，而是向量相对于某组基的描述。

8. 常见误区

  • 向量不只是箭头，箭头只是二维或三维中的可视化方式。
  • 坐标依赖基，同一个向量在不同基下可以有不同坐标。
  • 两个向量不一定能表示整个平面，只有不共线时才可以。
  • span 不是一个操作，而是所有线性组合形成的集合。

连接矩阵：为什么本章是矩阵乘法的前置条件

把两个向量作为列放在一起：

A = [ u  v ]

如果：

u = [u1, u2]
v = [v1, v2]

那么：

A = [[u1, v1],
     [u2, v2]]

矩阵乘向量：

A * [a, b]

其实就是：

a*u + b*v

本章最重要的一句话：线性组合 a*u + b*v 是线性代数的基本动作。用缩放和相加，从已有向量生成新的向量。

练习

概念题

  1. 为什么说“向量是箭头”只是一个直观模型，而不是向量的完整定义？
  2. 向量加法 u + v 的几何意义是什么？
  3. 数乘 a*v 中，a = 0、a > 0、a < 0 分别会发生什么？
  4. 用自己的话解释线性组合 a*u + b*v。
  5. span(u, v) 表示什么集合？
  6. 在二维平面中，两个非零向量什么时候只能 span 一条直线？
  7. 为什么“坐标”依赖于所选的基？

手算题

  1. 已知 u = [2, 1]，v = [-1, 3]，计算 u + v、2u、-3v、2u - v。
  2. 已知 u = [1, 2]，v = [3, 1]，计算 2u + 3v、-u + 4v、0.5u - 2v。
  3. 判断下面每组二维向量是否共线：[1, 2] 和 [2, 4]；[1, 0] 和 [0, 1]；[3, -6] 和 [-1, 2]；[2, 3] 和 [4, 9]。
  4. 已知标准基 e1 = [1, 0]，e2 = [0, 1]，把 [4, -2] 写成 a*e1 + b*e2 的形式。
  5. 已知 u = [2, 1]，v = [1, 2]，计算 3u - v，并解释这个结果表示从原点出发如何移动。

互动实验题

  1. 把 u 和 v 拖成大致共线。观察浅绿色采样区域发生了什么变化，并写下原因。
  2. 分别设置 a = 2, b = 0；a = 0, b = 2；a = 1, b = 1；a = 1, b = -1，记录绿色向量的位置变化。
  3. 把 u 和 v 拖成明显不共线，再把它们拖成共线。比较两种情况下 span 的差异。
  4. 固定 u 和 v，只改变 a、b。说明系数如何控制最终向量。

思考题

  1. 如果只允许使用一个非零向量和任意实数倍数，为什么最多只能到达一条直线？
  2. 如果两个二维向量不共线，为什么它们可以到达平面上的任意点？
  3. 如果把两个向量作为矩阵的两列，为什么矩阵乘 [a, b] 等价于 a*u + b*v？
  4. 为什么线性组合是理解矩阵乘法、线性方程组和线性变换的基础？

参考答案要点

概念题答案要点
  1. 向量的核心是可以相加和缩放；箭头只是二维/三维中的可视化方式。
  2. u + v 表示先走 u 再走 v，最终到达的位置。
  3. a > 0 保持方向缩放，a < 0 反向缩放，a = 0 得到零向量。
  4. a*u + b*v 表示沿 u 方向走 a 份，再沿 v 方向走 b 份。
  5. span 是所有可能线性组合组成的集合。
  6. 两个非零向量共线时，它们只提供一个方向。
  7. 坐标表示“相对于某组基走多少份”，换基后份数通常会变。
手算题答案要点
  1. u + v = [1, 4]，2u = [4, 2]，-3v = [3, -9]，2u - v = [5, -1]。
  2. 2u + 3v = [11, 7]，-u + 4v = [11, 2]，0.5u - 2v = [-5.5, -1]。
  3. 共线；不共线；共线；不共线。
  4. [4, -2] = 4*e1 - 2*e2。
  5. 3u - v = [5, 1]，表示从原点向右 5 个单位、向上 1 个单位。
互动实验与思考题答案要点
  1. 共线时所有组合都只能沿同一方向伸缩，所以采样区域退化为直线。
  2. a = 2, b = 0 只沿 u 走；a = 0, b = 2 只沿 v 走；a = 1, b = 1 是两者相加；a = 1, b = -1 是沿 u 走并抵消一部分 v 方向。
  3. 一个非零向量只能提供一个方向，任意倍数仍在同一条线上。
  4. 两个不共线向量提供两个独立方向，在二维平面中可以组合出任意位置。
  5. 矩阵的两列就是两个输出方向，乘以 [a, b] 等于分别按 a、b 缩放两列再相加。
