01. 向量与线性组合

本页从一个具体问题出发：如果只能沿几个方向移动，靠缩放和相加到底能到达哪里？先用交互建立直觉，再把这种动作写成定义、公式和练习。

学习路线 互动演示 知识讲解 连接矩阵 练习 答案要点 调研依据

本章要解决的问题

核心问题

给定两个方向 u 和 v，只允许做“缩放”和“相加”，我们能生成哪些向量？

  1. 对象：向量不是只会画箭头，它是可以相加、可以缩放的量。
  2. 动作：数乘改变走多少份，向量加法把几段移动接起来。
  3. 组合：a*u + b*v 是沿两个方向分别走 a 份和 b 份。
  4. 范围：span 是所有可能组合出的结果，也就是“能到达的地方”。
  5. 冗余：共线向量只提供一个方向，不共线向量提供两个独立方向。
  6. 坐标：当 u、v 是一组基时，[a,b] 表示相对这组基的走法。

知识讲解

1. 向量是什么

在二维平面里，向量可以画成从原点出发的箭头，例如 [3,2] 表示向右 3、向上 2。

但箭头只是可视化。更本质地说，向量是一种可以做两件事的对象：和另一个向量相加，被一个数字缩放。

2. 数乘

数乘回答“沿这个方向走多少份”。设 v = [v1,v2]，则：

a*v = [a*v1, a*v2]
  • a > 1：同方向拉长。
  • 0 < a < 1：同方向缩短。
  • a = 0：回到零向量。
  • a < 0：反方向走。

3. 向量加法

向量加法回答“先走一段，再走一段，最后在哪里”。设 u = [u1,u2]，v = [v1,v2]，则：

u + v = [u1 + v1, u2 + v2]

这不是两个数字列表的机械相加，而是把两次移动接成一次总移动。

4. 线性组合

把“缩放”和“相加”连起来，就得到线性组合：

a*u + b*v

它的含义是：沿 u 走 a 份，沿 v 走 b 份，然后把两段移动合并。

u = [2, 0]
v = [0, 3]
a = 1.5
b = 2
a*u + b*v = [3, 6]

5. Span

span 不是一次计算，而是所有可能结果组成的集合。

span(u, v) = 所有 a*u + b*v

一个非零向量的 span 是穿过原点的一条直线；两个不共线二维向量的 span 是整个二维平面。

6. 线性相关与线性无关

如果一个向量只是另一个向量的倍数，它没有带来新方向。这组向量线性相关（linearly dependent）。

u = [1, 2]
v = [2, 4]
v = 2u

看起来有两个箭头，实际上只有一条可走的直线。

7. 基与坐标

如果一组向量既能 span 整个空间，又没有冗余方向，那么它就是这个空间的一组基（basis）。

e1 = [1, 0]
e2 = [0, 1]
[x, y] = x*e1 + y*e2

坐标不是向量本身，而是“为了得到这个向量，每个基向量各走多少份”。

8. 常见误区

  • 向量不只是箭头，箭头只是二维或三维中的可视化方式。
  • 坐标依赖基；同一个几何向量，换一组基，坐标通常会变。
  • 两个向量不一定能表示整个平面；只有不共线且都不为零时才可以。
  • span 不是一个操作，而是所有线性组合形成的集合。
  • 画布里的浅绿色点是有限采样，不等于完整的无限集合。

连接矩阵：为什么本章是矩阵乘法的前置条件

把两个向量作为列放在一起：

A = [ u  v ]

如果：

u = [u1, u2]
v = [v1, v2]

那么：

A = [[u1, v1],
     [u2, v2]]

矩阵乘向量：

A * [a, b]

其实就是：

a*u + b*v

本章最重要的一句话：矩阵乘向量可以先理解为“用输入坐标当系数，把矩阵的列向量线性组合起来”。

练习

概念题

  1. 为什么“能相加、能缩放”比“画成箭头”更接近向量的本质？
  2. 向量加法 u + v 为什么可以理解成“先走 u，再走 v”？
  3. 数乘 a*v 中，a = 0、a > 0、a < 0 分别代表什么移动方式？
  4. 用自己的话解释线性组合 a*u + b*v。
  5. 为什么 span(u, v) 可以理解成“能到达的地方”？
  6. 为什么坐标不是向量本身，而是相对于某组基的描述？

手算题

  1. 已知 u = [2, 1]，v = [-1, 3]，计算 u + v、2u、-3v、2u - v。
  2. 已知 u = [1, 2]，v = [3, 1]，计算 2u + 3v、-u + 4v、0.5u - 2v。
  3. 判断下面每组二维向量是否共线：[1, 2] 和 [2, 4]；[1, 0] 和 [0, 1]；[3, -6] 和 [-1, 2]；[2, 3] 和 [4, 9]。
  4. 已知 u = [2, 1]，v = [1, 2]，找出 a,b，使 a*u + b*v = [5, 4]。
  5. 已知标准基 e1 = [1, 0]，e2 = [0, 1]，把 [4, -2] 写成 a*e1 + b*e2 的形式。

互动实验题

  1. 切到“2 数乘”，分别设置 a = 2、a = -1、a = 0，观察 a*u 的方向和长度。
  2. 切到“5 Span”，分别点击“不共线”“共线”“只有一方向”，比较浅绿色采样形状。
  3. 切到“6 基与坐标”，拖动目标 w。当 u、v 不共线时，观察页面给出的 a,b。
  4. 在“共线”预设下拖动 w，找一个能到达的目标和一个不能到达的目标。

思考题

  1. 如果只允许使用一个非零向量和任意实数倍数，为什么最多只能到达一条直线？
  2. 如果两个二维向量不共线，为什么它们可以到达平面上的任意点？
  3. 如果两个向量几乎共线，虽然理论上仍能 span 整个平面，为什么实际计算可能变得不稳定？
  4. 如果把两个向量作为矩阵的两列，为什么矩阵乘 [a, b] 等价于 a*u + b*v？

参考答案要点

概念题答案要点
  1. 箭头只是在二维/三维里的图像；向量真正重要的结构是可以相加和被数字缩放。
  2. 因为 u + v 把两段位移合成一段总位移，坐标上就是对应分量相加。
  3. a > 0 同方向走，a < 0 反方向走，a = 0 回到零向量。
  4. a*u + b*v 表示沿 u 走 a 份，沿 v 走 b 份，再合成最终位移。
  5. 因为 span 收集了所有可能的 a*u + b*v，也就是这些方向能生成的全部结果。
  6. 坐标回答的是“相对于这组基，每个基向量各用多少份”；换基后这组份数通常改变。
手算题答案要点
  1. u + v = [1, 4]，2u = [4, 2]，-3v = [3, -9]，2u - v = [5, -1]。
  2. 2u + 3v = [11, 7]，-u + 4v = [11, 2]，0.5u - 2v = [-5.5, -1]。
  3. 共线；不共线；共线；不共线。
  4. 解方程 2a + b = 5、a + 2b = 4，得到 a = 2、b = 1。
  5. [4, -2] = 4*e1 - 2*e2。
互动实验与思考题答案要点
  1. 数乘只改变沿同一方向走多少份；负数会反向，零会回到原点。
  2. 不共线时采样像一片区域；共线或只有一方向时采样退化为穿过原点的一条线。
  3. 不共线时任意目标 w 都有唯一的 a,b；共线时只有落在同一直线上的目标可到达。
  4. 一个非零向量的所有倍数仍在同一条线上，所以最多只能到达一条直线。
  5. 两个不共线向量提供两个独立方向，在二维平面中可以组合出任意位置。
  6. 几乎共线时两个方向提供的信息非常接近，为了到达某些点需要很大的正负系数互相抵消，计算对微小误差更敏感。
  7. 矩阵的两列就是两个输出方向，乘以 [a, b] 等于分别按 a、b 缩放两列再相加。
