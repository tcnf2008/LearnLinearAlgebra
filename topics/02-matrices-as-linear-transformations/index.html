02. 矩阵作为线性变换

本页把矩阵从“一张数字表”重新看成一个函数：输入一个向量，输出另一个向量。你会直接看到网格、基向量、普通向量和复合变换如何被同一组数字控制。

本章要解决的问题

核心问题

为什么一个 2x2 矩阵可以看成“把整个二维平面移动到新位置”的规则？

  1. 对象：矩阵不是先天为了做表格计算；在这里，它记录一个向量函数。
  2. 约束：线性变换必须保留加法和数乘，所以原点不动，网格线仍然直且均匀。
  3. 信息量：只要知道标准基 e1、e2 变到哪里，所有向量的去向就确定了。
  4. 计算：A[x,y] 是用 x、y 当系数，对矩阵两列做线性组合。
  5. 复合：先做 B 再做 A，总效果仍是一个线性变换，矩阵写成 AB。

知识讲解

1. 矩阵可以先看成函数

一个矩阵 A 作用在向量 v 上，得到新向量 Av。从函数视角看，输入是位置或移动，输出是变换后的位置或移动。

T(v) = A v

这种看法比“背乘法规则”更接近线性代数的核心：我们关心的是整个空间如何被一个规则移动。

2. 什么叫线性

线性变换必须同时满足两件事：

T(u + v) = T(u) + T(v)
T(kv) = kT(v)

几何上，这意味着原点保持不动，直线仍变成直线，平行且均匀间隔的网格仍保持这种结构。平移会移动原点，所以不是本章讨论的线性变换。

3. 两列决定整个变换

二维里任意向量都能写成 x*e1 + y*e2。如果知道 e1、e2 变到哪里，就知道所有向量变到哪里。

A = [[a, b],
     [c, d]]

A e1 = [a, c]
A e2 = [b, d]

所以矩阵的第一列是 e1 的落点，第二列是 e2 的落点。

4. 矩阵乘向量是列的线性组合

输入 [x,y] 的含义是“沿 e1 走 x 份，沿 e2 走 y 份”。线性变换后，这个走法不变，只是基向量的落点变了。

A [x, y] = x*[a, c] + y*[b, d]
         = [a*x + b*y, c*x + d*y]

这就是矩阵乘法公式背后的几何意义。

5. 矩阵乘法表示复合

如果先做 B，再做 A，对任意向量 v 都是：

v -> Bv -> A(Bv)

这个总效果仍然是一个线性变换，因此也可以用一个矩阵表示。这个矩阵记作 AB：

(AB)v = A(Bv)

顺序不能随意交换；AB 表示先 B 后 A。

6. 压扁、翻转与面积

det(A)=a*d-b*c 可以先粗略理解为单位正方形面积的缩放因子。

  • det(A) = 0：平面被压成一条线或一个点，信息丢失。
  • det(A) > 0：方向感保持，例如逆时针顺序仍是逆时针。
  • det(A) < 0：方向感翻转，好像空间被镜像了一次。

本章最重要的一句话：A[x,y] 不是神秘规则，而是“用输入坐标 x,y 去组合矩阵的两列”。

思考题

  1. 如果 det(A)=0，为什么通常无法从输出 Av 唯一找回输入 v？

参考答案要点

展开精简答案
  1. 因为任意二维向量都可写成 x*e1 + y*e2，线性变换会保留加法和数乘，所以 T(xe1 + ye2)=xT(e1)+yT(e2)。
  2. 平移会把原点移走，而线性变换必须满足 T(0)=0；也可以说它不满足 T(u+v)=T(u)+T(v)。
  3. A[3,2] = 3*[2,1] + 2*[-1,3] = [4,9]。
  4. B e1=[0,1]，B e2=[-1,0]，这是逆时针旋转 90°。
  5. 复合时先算 Bv，再算 A(Bv)；产品矩阵 AB 被定义成捕捉同一个总效果，所以 (AB)v 与 A(Bv) 一致。
  6. det(A)=0 时至少一个方向被压扁，多个输入可能落到同一个输出，信息丢失后就不能唯一反推。
