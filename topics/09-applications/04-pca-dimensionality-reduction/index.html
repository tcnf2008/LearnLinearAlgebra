09. 应用专题：PCA 降维

PCA 降维把高维数据重新换一组正交坐标：第一条坐标轴解释最大方差，第二条解释剩下的最大方差。保留前几个主成分，就能用更少坐标近似原始数据。

本专题要解决的问题

核心问题

三维甚至更高维数据能不能压成一两个坐标，同时尽量保留原来的主要差异？

  1. 中心化：先减去每个特征的均值，让 PCA 研究“围绕中心如何展开”。
  2. 协方差：用 Cov=X_c^T X_c/(n-1) 描述不同特征如何共同变化。
  3. 主成分：协方差矩阵的特征向量给出新的正交坐标轴。
  4. 解释方差：特征值越大，说明沿该方向的数据变化越强。
  5. 降维：只保留前 k 个主成分，把样本投影到低维坐标。
  6. 重建误差：被舍弃的主成分对应丢失的信息，表现为投影回原空间后的误差。

知识讲解

1. PCA 降维是什么

PCA 的目标不是任意删掉几个特征，而是重新构造一组正交坐标轴，让前几个坐标尽可能保留数据变化。

高维样本 -> 主成分坐标 -> 保留前 k 个坐标

如果数据本来接近一条线或一个平面，PCA 常常能用一维或二维坐标保留大部分结构。

2. 为什么必须中心化

PCA 研究的是样本相对均值的变化方向。中心化把每个特征减去均值：

X_c = X - mean(X)

如果不中心化，第一主成分可能被数据整体离原点的位置主导，而不是被样本之间的差异主导。

3. 协方差矩阵在说什么

协方差矩阵记录特征的共同变化：

Cov = X_c^T X_c / (n - 1)

对角线是每个特征自己的方差，非对角线是两个特征是否同向变化。PCA 就是在这个矩阵里找最大变化方向。

4. 主成分为什么是特征向量

沿单位方向 v 投影后的方差是 v^T Cov v。要找方差最大的单位方向，会得到特征向量问题：

Cov v_i = λ_i v_i

λ_i 是沿第 i 个主成分的方差，按从大到小排序。

5. 降维和重建

保留前 k 个主成分矩阵 V_k 后，低维坐标和重建为：

Z = X_c V_k
X_hat = Z V_k^T + mean(X)

如果舍弃的特征值很小，重建误差也会较小；如果舍弃方向仍有重要结构，降维就会丢信息。

6. 标准化什么时候重要

PCA 对特征尺度敏感。一个单位很大的特征会产生很大的方差，从而主导第一主成分。

如果特征单位不同，比如收入、年龄、点击次数混在一起，通常先标准化；如果单位相同且方差大小本身有意义，则不一定要标准化。

一句话总结：PCA 通过协方差矩阵找到数据最主要的正交变化方向，降维就是保留前几个方向并舍弃弱方向。

思考题

  1. 如果一个低方差方向恰好对分类非常关键，PCA 会保留它还是舍弃它？这说明 PCA 适合什么、不适合什么？
  2. PCA 降维和 SVD 图像压缩有什么共同点？

参考答案要点

展开精简答案
  1. 中心化移动坐标原点到样本均值，不改变样本之间的相对距离；它让 PCA 研究数据如何围绕中心展开。
  2. PCA 是无监督方法，只看输入特征的方差结构，不看标签；最大方差方向不一定最有预测价值。
  3. 第一主成分是 x 轴方向；保留 1 维解释方差比例为 9/(9+1)=90%。
  4. 解释方差比例为 (6+3)/(6+3+1)=90%。
  5. 低方差但关键的分类方向可能被 PCA 舍弃；PCA 适合压缩、去噪、可视化，不适合直接保证监督任务效果。
  6. 两者都在用少数正交方向近似原始矩阵；PCA 保留数据矩阵的主要方差方向，SVD 图像压缩保留像素矩阵的主要低秩结构。
