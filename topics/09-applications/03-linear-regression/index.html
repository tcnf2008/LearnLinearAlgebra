09. 应用专题：线性回归

线性回归把“用输入预测输出”的问题写成矩阵近似：选择参数 θ，让预测向量 Xθ 尽量接近真实观测 y。最小二乘解就是把 y 投影到设计矩阵 X 的列空间上。

本专题要解决的问题

核心问题

数据点通常不在同一条直线上，为什么线性代数仍然能给出一条“最好”的预测直线？

  1. 建模：用 y ≈ θ0 + θ1x 表示输入和输出之间的线性关系。
  2. 矩阵化：把所有样本写成 Xθ ≈ y，其中 X 的第一列是 1，第二列是输入 x。
  3. 误差：残差 e=y-Xθ 表示每个样本预测错了多少。
  4. 目标函数：最小化平方误差和 ||y-Xθ||^2，大误差会被更重惩罚。
  5. 正规方程：最优解满足 X^T(y-Xθ)=0，残差与列空间正交。
  6. 泛化边界：最小训练误差不等于最好预测能力，异常点、非线性和特征尺度都会影响模型。

知识讲解

1. 线性回归是什么

最简单的一元线性回归假设输出可以由输入的线性函数近似：

y_hat = θ0 + θ1 x

θ0 是截距，表示 x=0 时的预测值；θ1 是斜率，表示输入增加 1 个单位时预测值改变多少。

2. 为什么写成 Xθ≈y

把所有样本排成矩阵：

X = [[1, x1],
     [1, x2],
     ...
     [1, xn]],     θ = [θ0, θ1]^T

那么所有预测可以一次写成 y_hat=Xθ。这让回归从“画一条线”变成“在列空间里找最接近 y 的向量”。

3. 为什么用平方误差

残差是 e=y-Xθ。最小二乘选择让 ||e||^2 最小的参数：

minimize ||y - Xθ||^2

平方误差可导、容易计算，并且会更重惩罚大误差；这也意味着异常点会产生很大影响。

4. 正规方程从哪里来

最优预测 Xθ* 是 y 在列空间 Col(X) 上的投影。投影误差必须垂直于整个列空间：

X^T(y - Xθ*) = 0

展开后得到正规方程 X^T X θ* = X^T y。

5. R² 表示什么

R² 比较模型误差和“只预测平均值”的误差：

R² = 1 - SSE / SST

它越接近 1，说明模型解释了越多输出变化；如果模型比均值预测还差，R² 甚至可以是负数。

6. 工程里为什么不用直接求逆

理论上可以写 θ=(X^TX)^{-1}X^Ty，但工程中通常不用显式求逆。若特征列近似相关，X^TX 会病态，QR、SVD 或带正则项的岭回归更稳。

本页用一元回归建立直觉；真实应用会扩展到多特征、特征缩放、正则化、训练/测试划分和误差分析。

一句话总结：线性回归不是“随手画线”，而是在设计矩阵的列空间里找离观测向量最近的投影点。

练习

概念题

  1. 为什么线性回归的残差向量在最优点要与 X 的每一列正交？
  2. 为什么带异常点的数据会明显拉动最小二乘直线？

手算题

  1. 数据点为 (0,1)、(1,3)、(2,5)。直接写出最优直线，并说明为什么残差为 0。
  2. 若某模型 SSE=20，只预测均值时 SST=50，求 R²。

互动实验题

  1. 选择“近似线性”，点击“设为最优”，观察 |X^T e| 是否接近 0。
  2. 选择“带异常点”，拖动最右侧异常点，观察最优线、SSE 和 R² 如何变化。
  3. 选择“非线性趋势”，比较最小二乘线的低 SSE 是否足以说明模型形式正确。

程序题

  1. 运行 .venv/bin/python topics/09-applications/03-linear-regression/demo.py，查看正规方程、numpy.linalg.lstsq 和梯度下降得到的参数是否一致。
  2. 在脚本里增大异常点的 y 值，观察最小二乘参数和 RMSE 的变化。

思考题

  1. 为什么训练误差小不一定代表未来预测好？
  2. 如果两个特征几乎一样，为什么 X^TX 可能变得难以稳定求解？

参考答案要点

展开精简答案
  1. 如果残差在列空间方向上还有分量，就可以沿该方向继续移动预测向量，让误差更小；最优投影点的残差必须垂直于列空间。
  2. 平方误差会把大残差放大，异常点的损失贡献可能超过很多普通点。
  3. 最优直线是 y=1+2x，三个点全部在直线上，所以每个残差都是 0。
  4. R²=1-20/50=0.6，表示模型解释了约 60% 的总变化。
  5. 非线性数据中，即使线性模型找到最小 SSE，也可能系统性低估或高估某些区域；这说明模型形式有偏差。
  6. 训练误差只评价已有样本，未来样本可能受噪声、采样偏差或分布变化影响；过度贴合训练数据也会损害泛化。
  7. 两个几乎相同的特征让列空间方向接近重合，X^TX 接近奇异，小数据扰动就会导致参数大幅变化。
