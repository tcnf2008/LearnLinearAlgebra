03. 线性方程组与解空间

本页把方程组从“代数题”重新看成“约束的交集”：每个线性方程筛出一条直线，方程组要找同时满足所有约束的点。你会看到唯一解、无解、无穷多解，以及消元为什么没有改变真正的问题。

本章要解决的问题

核心问题

线性方程组不是在“凑数字”，而是在找哪些向量同时满足一组线性约束。

  1. 一个方程：a*x + b*y = r 在二维平面里通常是一条直线。
  2. 多个方程：方程组的解是所有约束线的交集。
  3. 三种结果：两条线相交是唯一解；平行分离是无解；完全重合是无穷多解。
  4. 矩阵写法：Ax=b 把多条约束压缩成一个统一问题。
  5. 消元：行变换用等价约束替换原约束，让交集更容易看出来。
  6. 解空间：齐次方程 Ax=0 的解是子空间；非齐次方程若有解，解集是一个平移后的子空间。

知识讲解

1. 一个方程是一条约束

a*x + b*y = r 不是让你先想算法，而是在平面中筛选点：所有满足它的 [x,y] 形成一条直线。

2x + y = 5

这句话的意思是：只要一个点的横坐标和纵坐标满足这个关系，它就在这条约束线上。

2. 方程组求交集

两个方程就是两个筛子。一个点必须同时通过两个筛子，才是方程组的解。

a1*x + b1*y = r1
a2*x + b2*y = r2

所以二维里最自然的几何图像就是两条线的交点、平行分离或完全重合。

3. 矩阵形式 Ax=b

把未知量放进向量，把系数排成矩阵，就得到统一写法：

A = [[a1, b1],
     [a2, b2]]
x = [x, y]
b = [r1, r2]

Ax = b

这不是换符号而已。它让我们能用矩阵的列空间、秩、零空间来理解所有方程组。

4. 三种解对应三种几何

  • 唯一解：两条线相交于一个点，矩阵有两个独立方向。
  • 无解：约束互相矛盾，例如两条平行线要求同一个点在两个不同位置。
  • 无穷多解：约束重复或不够多，留下整条线甚至整个平面。

5. 高斯消元为什么有效

消元不是改变答案，而是把约束换成更容易读的等价约束。典型行操作包括交换两行、某行乘非零数、用一行加减另一行的倍数。

R2 <- R2 - m*R1

如果一个点同时满足原来的两行，它也满足替换后的行；反过来结合保留的那一行，也能恢复原约束。因此解集不变。

6. 解空间与零空间

齐次方程 Ax=0 的解集一定包含零向量，并且对加法和数乘封闭，所以它是一个子空间，也叫 null space。

非齐次方程 Ax=b 如果有一个特解 xp，全部解可以写成：

x = xp + z, 其中 Az = 0

所以非齐次解集通常不是穿过原点的子空间，而是零空间平移后的集合。

本章最重要的一句话：解方程组就是找约束的共同部分；消元只是把同一个共同部分改写得更容易看。

少量练习

概念题

  1. 为什么一个二维线性方程通常对应一条直线，而两个方程的解对应两条线的交集？
  2. 为什么 Ax=0 的解集是子空间，而 Ax=b 的解集不一定是子空间？

手算题

  1. 解方程组：x + y = 3，x - y = 1。
  2. 判断方程组 x + 2y = 4，2x + 4y = 9 有无解，并说明几何原因。

互动实验题

  1. 依次点击“唯一解”“无解”“无穷多解”，观察右侧的 det(A) 和画布里的线关系。
  2. 切到“4 消元”，观察绿色虚线如何替代第二个方程，但不改变交点。

思考题

  1. 如果消元后出现 0 = 5，为什么这说明原方程组无解？

参考答案要点

展开精简答案
  1. 一个方程筛出所有满足同一线性关系的点；两个方程同时成立，就是取两条筛选结果的共同部分。
  2. Ax=0 的解对相加和数乘封闭；Ax=b 若 b != 0，通常不包含零向量，所以通常不是子空间。
  3. 相加得 2x=4，所以 x=2；代回得 y=1。
  4. 无解。第二个左边是第一个左边的 2 倍，但右边不是 2 倍；几何上是两条平行但不同的直线。
  5. det(A) != 0 时两行方向不平行，通常唯一交点；det(A)=0 时可能平行分离或重合。
  6. 0=5 没有任何 x,y 能满足；它是矛盾约束，因此原系统无解。

教学组织依据

本页采用“约束线 -> 交集 -> 矩阵形式 -> 消元 -> 解空间”的顺序，先让读者知道方程组在找什么，再解释算法为什么有效。

  • 概念校准：消元和增广矩阵如何把线性系统变成等价系统，参考 MIT OCW: Elimination with Matrices。
  • 直觉线索：把 Ax=b 看成“找哪个输入落到目标向量”，并连接 column space 与 null space，参考 3Blue1Brown: Inverse matrices, column space, and null space。
  • 练习组织：从方程组翻译到矩阵方程 Ax=b，参考 Khan Academy: Representing linear systems with matrix equations。
