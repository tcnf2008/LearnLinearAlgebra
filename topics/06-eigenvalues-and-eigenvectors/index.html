06. 特征值与特征向量

本页从一个简单问题出发：矩阵会改变大多数向量的方向，但有没有一些方向只被拉长、压短或反向，而不被转弯？这些方向就是 eigenvector，对应的缩放倍数就是 eigenvalue。

本章要解决的问题

核心问题

在一个线性变换里，哪些方向经过变换后仍然落在同一条直线上？

  1. 对象：特征向量是被矩阵作用后不改变方向的非零向量。
  2. 倍数：特征值记录这个方向被放大、缩小、反向或压成零的程度。
  3. 公式：Av = λv 表示输出 Av 和输入 v 共线。
  4. 求法：要有非零解，(A-λI)v=0 必须把空间压扁，所以 det(A-λI)=0。
  5. 意义：如果能找到足够多独立特征向量，复杂变换可以拆成沿若干方向的独立缩放。
  6. 限制：有些矩阵在实平面里没有真实特征方向，例如纯旋转 90°。

知识讲解

1. 特征向量是什么

矩阵通常会同时改变向量的长度和方向。特征向量是例外：它被矩阵作用后仍然在原来那条直线上。

Av = λv, v != 0

这里 v 是 eigenvector，λ 是 eigenvalue。v 不能是零向量，因为零向量没有方向。

2. 特征值表示什么

λ 是沿特征方向的缩放倍数。

  • λ > 1：沿同方向拉长。
  • 0 < λ < 1：沿同方向压短。
  • λ < 0：反向，并按绝对值缩放。
  • λ = 0：这个方向被压到零向量。

3. 为什么看 det(A-λI)

把 Av=λv 移到同一边：

(A - λI)v = 0

要找到非零 v，矩阵 A-λI 必须把某个方向压成零，也就是不可逆：

det(A - λI) = 0

这就是特征方程。二维里它是一个二次方程。

4. 为什么有些矩阵没有真实特征方向

纯旋转 90° 会把每个非零向量转到垂直方向。没有任何真实方向能保持在同一条直线上。

代数上，对应特征方程没有实根；如果扩展到复数，会出现复特征值，但二维实平面里看不到真实特征向量。

5. 对角化为什么有用

如果二维矩阵有两条独立特征方向，就可以把任意向量分解到这两条方向上。矩阵作用时，每个分量只按自己的特征值缩放。

A = P D P^-1

P 的列是特征向量，D 的对角线是特征值。这样 A^k 的效果就容易理解：每个特征方向缩放 λ^k。

6. 和前几章的连接

从矩阵变换看，特征向量是最稳定的方向。从线性方程组看，求特征向量是在求 (A-λI)v=0 的 null space。

这也解释了为什么本章依赖前面的概念：矩阵作为变换、行列式表示压扁、零空间表示被映到零的方向。

本章最重要的一句话：特征向量不是“特殊数字列表”，而是矩阵变换中不被转弯的方向；特征值是这个方向上的缩放倍率。

少量练习

概念题

  1. 为什么定义 Av=λv 时要求 v != 0？
  2. 如果一个矩阵是纯旋转 90°，为什么在实平面里没有特征向量？

手算题

  1. 设 A=[[3,0],[0,2]]，判断 [1,0]、[0,1]、[1,1] 哪些是特征向量，并给出特征值。
  2. 设 A=[[2,1],[1,2]]，验证 [1,1] 和 [1,-1] 是特征向量。

互动实验题

  2. 点击“旋转 90°”，观察页面如何提示“没有真实特征方向”。

思考题

  1. 为什么反复应用矩阵时，主特征值绝对值较大的方向常常会逐渐“显出来”？

参考答案要点

展开精简答案
  1. 零向量没有方向，而且任何 λ 都满足 A0=λ0，会让定义失去区分能力。
  2. 旋转 90° 会把每个非零向量变到垂直方向，不可能和原向量共线。
  3. [1,0] 是特征向量，特征值 3；[0,1] 是特征向量，特征值 2；[1,1] 不是，因为 A[1,1]=[3,2] 不与 [1,1] 共线。
  4. A[1,1]=[3,3]=3[1,1]；A[1,-1]=[1,-1]=1[1,-1]。
  5. 把初始向量分解到特征方向上后，每次应用矩阵都会把各分量乘以对应特征值。绝对值更大的特征值经过多次乘法增长更快，所以方向更明显。
